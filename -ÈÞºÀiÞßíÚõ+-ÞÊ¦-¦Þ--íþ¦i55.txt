- functional test ดูคอมเม้นได้เลย
- เพิ่ม name ในบาง tag ในบาง template เพื่อมาใช้กับ test
- แก้ template บังคับต้องใส่ชื่อวิชา (เพิ่ม required เข้าไปใน tag)
- แก้ views.py พวก render พิมพ์เกินมาทุกอัน หาตั้งนานว่า error ตรงไหน ฮา
- แก้ operand ใน views.py ย้ายใน except ออกมาเป็น default
- ถ้าเป็นแบบเก่า พอหลุดไป except แล้วค่ามันจะเพี้ยน
- แก้ else เป็นเช็คค่า subject แทน เพราะถ้าเป็นแบบใช้ else มันจะทำงานเมื่อ try ไม่มี error เท่านั้น แปลอีกทีคือ ตรงกรอกข้อมูลทุกช่อง

ที่ยังเหลือ
- unit test ไปลองงมๆดูก่อน ทางนี้ก็ยังไม่ได้ทำ ฮา
- ลิงค์กลับหน้าแรกมันจะ error เพราะติดเรื่อง log in ค่อนข้างจะดูวุ่นวาย ติดไว้ก่อน55
